Это плагин по разбиению полилинии на пикетажи. Для появления пикетажей в проекте QGIS необходимо заранее создать слой "Пикеты". В нем появятся точки на расстоянии 100 м. Система координат слоя - прямоугольная. Для запуска модуля, в проекте сначала выделяется слой с полилиниями, затем запускаем модуль.
